サインイン・ログインの機能について

ログインはGoogleAuthで可能（実装済み）
　一度ログイン情報は保持しておいて、次回以降はGoogleアカウント選択不要
　ログオフした場合は再度Googleアカウントの選択が必要
ログインはメールアドレス（パスワードで可能）
　

メールアドレスでのログイン用に、サインインの画面を準備
メールアドレス登録後、メールアドレスに対して、URLを案内
利用者はURL押下して、その画面で、パスワード設定（デフォルトはメールアドレスでOKです）
その後は、上記のメールアドレスと、パスワードでログイン可能にする

メールあどれはResend取得済み
.env
FERNET_KEY
RESEND_API_KEY
