# 実行プラン（XML）の取得と投稿方法

`qps5` で実行プランを共有するには、SQL Server Management Studio (SSMS) から **XML 形式のプランデータ**を取得する必要があります。以下の手順に従ってください。

## ステップ 1: SSMS で実行プランを表示する

1.  SSMS でクエリウィンドウを開き、対象の SQL クエリを入力します。
2.  ツールバーの **[実際の実行プランを含める]** アイコン（または `Ctrl + M`）をクリックして有効にします。
3.  クエリを実行（`F5`）します。
4.  結果画面の下部に表示される **[実行プラン]** タブをクリックします。



## ステップ 2: XML データを表示・コピーする

1.  グラフィカルな実行プラン上の**何もないところ（背景部分）を右クリック**します。
2.  コンテキストメニューから **[実行プラン XML の表示...]** を選択します。
3.  新しいウィンドウに XML コードが表示されます。
4.  `Ctrl + A` で全選択し、`Ctrl + C` でコピーします。

> [!TIP]
> **ファイルから取得する場合**
> すでに保存済みの `.sqlplan` ファイルがある場合は、そのファイルをテキストエディタ（メモ帳や VS Code）で開き、中身のテキストをすべてコピーしてください。

## ステップ 3: qps5 に貼り付けて投稿する

1.  `qps5` の [新規投稿] ページへ移動します。
2.  **[Query Plan XML]** 欄に、コピーした XML データをそのまま貼り付けます。
3.  タイトルやコメント（Markdown対応）を入力します。
4.  **公開設定** を選択します：
    * **全体公開**: 一覧に掲載され、誰でも検索・閲覧できます。
    * **限定公開**: 一覧には載りません。発行される専用URLを知っている人のみ閲覧可能です。
5.  **[投稿する]** ボタンをクリックして完了です。

---

## 注意事項
* **個人情報の保護**: 実行プラン XML には、テーブル名、インデックス名、サーバー名、ユーザー名などが含まれる場合があります。機密性の高い情報は、投稿前に必要に応じてマスキング（置換）することをお勧めします。
* **ファイルサイズ**: 極端に巨大な実行プラン（数MB以上）は、表示に時間がかかったり、ブラウザが重くなる場合があります。
