# ゲスト投稿の制限について

`qps5` はログインなし（ゲスト）の状態でもプランの投稿や共有が可能ですが、ログインユーザーとは利用できる機能の範囲が異なります。

## ゲストができること・できないこと

| 機能 | ゲストユーザー | ログインユーザー |
| :--- | :--- | :--- |
| **プランの新規投稿** | ✅ 可能 | ✅ 可能 |
| **公開設定の選択** | 🌐 Public / 🔗 Unlisted | 🌐 Public / 🔗 Unlisted / 🔒 Private |
| **投稿後の編集・削除** | ⚠️ **一時的のみ**（トークンが必要） | ✅ いつでも可能 |
| **コメントの投稿** | ✅ 可能 | ✅ 可能 |
| **自分のコメントの編集** | ❌ 不可 | ✅ 可能 |
| **投稿履歴の管理** | ❌ 不可 | ✅ マイページで管理可能 |
| **非公開設定 (Private)** | ❌ 不可 | ✅ 可能 |

---

## ゲストによる「編集・更新」の仕組み

ゲスト投稿の場合、ブラウザを閉じたりキャッシュをクリアしたりすると、通常はその投稿の編集権限を失います。これを補完するために **「編集トークン（Edit Token）」** という仕組みを採用しています。

### 1. 編集トークンの発行
プランをゲストとして投稿した直後の画面に、**「このプランを編集するための専用URL」** または **「トークン」** が表示されます。
* **重要**: このURLを紛失すると、ゲスト本人が後から内容を修正したり、期限を延ばしたり、削除したりすることができなくなります。必ずメモ等で控えてください。

### 2. 編集の有効期限
セキュリティ上の理由により、ゲストによる編集可能期間は投稿から一定期間（例：24時間以内など、システム設定による）に限定される場合があります。

---

## ログイン（アカウント登録）を推奨するケース

以下のような用途で `qps5` を利用される場合は、Google認証等によるログインをお勧めします。

* **長期的な管理**: 過去に投稿したプランを一覧で確認し、いつでも編集・削除したい場合。
* **完全な非公開**: URLを知っている人にも見せたくない、自分専用のメモとして保存したい場合（Private設定）。
* **コメントの修正**: 誤字脱字などを後から直したい場合。
* **Admin（管理者）への昇格**: 組織内でのプラン分析を主導し、不適切なコメントの削除権限などが必要な場合。

---

## 注意事項
* ゲスト投稿されたプランであっても、**利用規約に反する内容**や、著しくサーバーに負荷をかけるXMLが投稿された場合は、管理者の判断で予告なく削除・非アクティブ化することがあります。
